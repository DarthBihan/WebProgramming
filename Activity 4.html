Activity 4: Semantic Elements Comparison

Div vs Span

Div: A block-level element, used to group content and structure layouts. It starts on a new line and takes up the full width of its container.

Span: An inline element, used to style or group content within a block of text.

Inference: Div creates block-level sections, useful for layout purposes, while span is used for inline styling or grouping within text.

Div vs Section

Div: A generic container for content without semantic meaning.

Section: A semantic container for logically grouped content, such as chapters or sections in a document.

Inference: Div is non-semantic and purely structural, while section is semantic grouping of related content.

Paragraph vs Article

P: Represents a single block of text.

Article: A semantic container for self-contained content, such as blog posts or news stories.

Inference: Paragraph is for general text blocks, while article is for standalone, reusable, or exportable content.

Navigation vs Anchor

Nav: A semantic container for navigation links.

A: An inline element to create hyperlinks.

Inference: Nav is for groups of navigation-related links, while anchor creates individual hyperlinks.

Figure vs Image

Figure: Groups media content (like images) with captions or descriptions.

Img: Displays an image.

Inference: Figure is used to group an image with related text (like captions), while image is solely for displaying images.

Label vs Legend

Label: Associates descriptive text with a form input.

Legend: Provides a caption for a <fieldset> grouping.

Inference: Label binds text to a specific form input, whereas legend provides context to grouped form elements.

Strong vs Bold vs Emphasis

Strong: Indicates importance semantically.

Bold: Makes text bold visually but lacks semantic meaning.

Em: Conveys subtle emphasis, often italicized by default.

Inference: Strong is semantically important, bold is visually bold, and emphasis is for subtle stress or italics.
